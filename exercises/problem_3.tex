En \LaTeX, es puedo hacer una lista enumerada:
\begin{enumerate}
\item Primer elemento
\item Segundo elemento
\item Tercer elemento
\end{enumerate}

Y también, puedo hacer una lista no enumerada:
\begin{itemize}
\item Primer elemento
\item Segundo elemento
\item Tercer elemento
\end{itemize}
